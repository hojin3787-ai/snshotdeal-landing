## 🆘 문제 해결

### "소유권을 확인할 수 없습니다"
1. 메타 태그가 `<head>` 안에 있는지 확인
2. 실제 배포된 사이트에서 소스 보기로 태그 존재 확인
3. 브라우저 캐시 지우고 재시도
4. 코드를 정확히 복사했는지 재확인 (공백, 따옴표 등)

### "Sitemap을 가져올 수 없습니다"
1. https://xn--sns-h84mk60k.com/sitemap.xml 직접 접속 확인
2. XML 형식 오류 없는지 확인
3. robots.txt에서 sitemap 차단 여부 확인
4. 24시간 후 다시 확인 (서버 동기화 시간)

### "색인이 생성되지 않습니다"
1. 콘텐츠가 충분한지 확인 (최소 300단어 이상 권장)
2. 중복 콘텐츠가 아닌지 확인
3. 페이지 속도 문제가 없는지 확인
4. 2-4주 더 대기 (특히 신규 사이트)
