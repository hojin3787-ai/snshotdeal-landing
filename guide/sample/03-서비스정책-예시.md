# 서비스정책 — 예시 모음

> 본문(가이드)에서는 예시를 제외하고 형식/원칙을 중심으로 설명합니다. 아래는 참고용 예시입니다.

## 에너지 예시(형식)
```markdown
### 획득
- 충전: 30초당 +1 (일일 제한)
- 출석: 시간대별 지급 (일일 제한)
- 가입 보너스: 최초 1회

### 사용
- 응모 1회당 에너지 N개 소모
- 콘텐츠 1회당 에너지 N개 소모
```

## 포인트 예시(형식)
```markdown
### 획득
- 콘텐츠 보상, 초대 보상

### 사용
- 상점에서 기프티콘 구매

### 소멸
- 마지막 적립일 기준 N일 후 소멸
```

## 광고/응모/출석 예시(형식)
```markdown
- 광고(보상형/전면/배너/MREC) 노출 위치/조건/타임아웃/재로딩 정책
- 응모: 클릭 즉시 응모, 일정 간격 광고 노출, 당첨자 1명 보장
- 출석: 시간대별 1회, 일일 최대 N회, 00:00 리셋
```

> 실제 수치와 규칙은 해당 프로젝트의 서비스정책 본문에 따릅니다.
